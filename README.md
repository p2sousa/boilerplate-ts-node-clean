# boilerplate-ts-node-clean

## Como executar (Local):
### Requesitos:
1. [NodeJs](https://nodejs.org/en/)
2. [Yarn](https://classic.yarnpkg.com/lang/en/)

---

# Testes

O Projeto possui testes unitarios e de integração:

Testes unitarios
```shell
$ yarn test:unit
```

Testes de integração
```shell
$ yarn test:int
```

Rodando as 2 suites junto
```shell
$ yarn test
```
